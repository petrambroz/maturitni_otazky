Základní komponenty, parametry, zkratky

Základní deska

  • Propojuje jednotlivé komponenty, zajišťuje komunikaci mezi nimi
  • Umožňuje připojení dalších zařízení
  • Motherboard, MB
  • Patice pro CPU
  • Ram sloty
  • Chipset (musí podporovat procesor)
  • Sběrnice pro GPU
  • Porty na zadní straně
  • SATA konektory

Procesor

  • „Mozek počítače“
  • CPU
  • Intel, AMD, …
  • Počet jader + vláken
  • Taktovací frekvence
  • Patice
  • Cache

Operační paměť

  • Rychle dostupná paměť
  • RAM
  • Kapacita
  • Frekvence
  • Časování
  • Typ sběrnice

Grafická karta

  • Zpracovává obraz pro monitor
  • GPU
  • NVidia, AMD
  • Buď integrovaná v procesoru (levnější PC) nebo samostatná
  • Frekvence
  • GDDR paměť
  • Počet jader

Zdroj

  • Napájí veškeré komponenty
  • Mění 230v napětí na 12v/5v
  • PSU (power supply unit)
  • Maximální výkon
  • Účinnost

HDD, SSD

  • Uchovává informace
  • Kapacita (HDD větší a levnější)
  • Rychlost čtení a zápisu (HDD menší, SSD větší)
  • Připojení většinou přes SATA (případně M.2 SSD)
