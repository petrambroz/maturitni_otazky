  • proměnná – označení, definice proměnné, typ
  • statický a dynamický typový systém
  • přehled základních datových typů
  • operace %, –, +, *, /, cos, sin, abs, …

Uchování dat

  • Data můžeme uchovávat v proměnné

Označení, definice, typ

  • Proměnná má své jméno
  • Jsou rozlišena velká a malá písmena
  • Definice proměnné – v Pythonu je definována ve chvíli, kdy k ní přiřadíme nějakou hodnotu

cislo = 10
slovo = "slovo"

Statický a dynamický systém

Statický

  • Používá se např. v Javě, C, C++, C#, TypeScriptu
  • Při deklaraci proměnné je nutné uvést, jakého datového typu bude, tento typ pak nelze snadno změnit
  • int a = 10;
    • a je celé číslo
  • float pi = 3.14
    • pi je desetinné číslo
  • String slovo = "ahoj";
  • slovo je řetězec znaků

Dynamický

  • Využívá se v Pythonu, JavaScriptu
  • Stačí proměnné přiřadit hodnotu, Python si sám vybere nejvhodnější datový typ (→ možná chybovost)
  • a = 10
    • a je celé číslo
  • pi = 3.14
    • pi je desetinné číslo
  • slovo = "ahoj"
  • slovo je řetězec znaků
  • Lze však ručně datový typ nastavit
  • tri = str(3)
  • tri je sice 3, ale pouze jako znak, ne číslo → nelze použít pro výpočty

Datové typy

  • Příklad v Javě:
  • int – celé číslo
  • float – desetinné číslo
  • String – řetězec znaků
  • char – jeden znak
  • boolean – hodnota True/False
  • list
  • dictionary

Operace

  • +, -, *, / – klasika sčítání, odčítání, násobení, dělení
  • % – tzv. modulo → zbytek při dělení

a = 10
b = 3
vysledek = 10%3
print(vysledek)
  • Výstup – 1
    • 10/3 = 3, zbytek 1

Sin, cos

  • Vypočítá sinus a cosinus čísla
  • K využití je nutné importovat knihovnu math
    • import math
  • Syntaxe: a = math.sin(0.5)
  • Díky knihovně math můžeme využít např. i math.pi() – hodnota π

Abs

  • Absolutní hodnota daného čísla
  • Syntaxe: a = abs(-4.5)
  • a se tedy rovná 4.5
