Funkce, procedury

  • Jsou to samostatné bloky kódu, jsou definované předtím než jsou volány, mají svůj název
  • Funkce – vrací nějakou hodnotu
def funkce_mocnina(cislo):
    mocnina = cislo ** 2
    return(mocnina)
vstup = int(input("zadej cislo: "))
print(funkce_mocnina(vstup))
  • Procedura – nic nevrací (v Javě void)
def procedura_mocnina(cislo):
    mocnina = cislo ** 2
    print(mocnina)
vstup = int(input("zadej cislo: "))
procedura_mocnina(vstup)
  • Výstup je v obou stejný, procedura je v tomto případě o něco jednodušší, ale neumožnila by nám další práci s umocněným číslem, pouze ho vyprintuje
  • Funkce a procedury můžeme volat v programu kolikrát chceme
    • Např. potřebujeme na více místech vypočítat mocninu čísla a nechceme vždy vypisovat výpočet umocnění

Předání parametru

  • V definice funkce musíme říct, jaké parametry (proměnné) jí předáme
    • V těle funkce se pak budou jmenovat tak, jak jsou pojmenované v definici
  • Při volání funkce musíme předat všechny parametry, které funkce vyžaduje

Princip, využití rekurze

  • Funkce volá sebe sama ještě před tím, než je dokončena

Fibonacciho posloupnost

def fib(n):
    if n == 0:
        return 0
    elif n == 1:
        return 1
    return fib(n-1)+fib(n-2)
for i in range(10):
    print(fib(i))

Faktoriál

def factorial(x):
    if x == 0:
        return 1
    elif x > 0:
        return x * factorial(x - 1)
print(factorial(int(input())))
